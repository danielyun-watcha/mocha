## 풀 리포트 모드

```
# 📊 [도메인] EDA 리포트
[메인 파일 · n_rows · 분석 일시]

## 📅 데이터 개요
| 항목 | 값 |
|---|---|
| 유저 | ... |
| 콘텐츠 | ... |
| Sparsity | ... |

[F1 이미지]

## 📈 시간·꼬리 분포

### ⏱️ 시간 분포
- 일평균 인터랙션: ...

[F2 이미지]

### 📉 꼬리 분포
- 상위 1% → 전체 X%
- 상위 5% → 전체 Y%

[F3 이미지]
```

## Q&A 모드

질문 의도에 맞는 표 1-2개 + 집계 기준 + 인사이트만. Appendix 생략.

```
# 🐼 [질문]
[도메인 · 파일 · 일시]

## 📅 결과
[해당 표 1-2개]

## 📊 집계 기준
[full과 동일]

## 💡 주요 인사이트
[관련 suggestion 3단]
```

## 인사이트 3단 구조

```
✅ **{한 줄 결론 — analysis_suggestions에서 그대로}**
- {수치 부연 — case_studies/overview 인용}
- {비교 맥락 부연 — "X 대비 N배"}
- {해석 — "이는 ... 로 해석됨" 또는 "모델에서 ... 가능"}
```

부연·해석은 Python이 빈 슬롯으로 두고 Claude가 JSON 컨텍스트 보고 채운다.
