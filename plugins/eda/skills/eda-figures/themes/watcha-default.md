# Watcha Default Theme

기본 테마. 라이트 배경 + 친화적 톤. Anthropic brand-guidelines 스타일을 따른다.

## Primary Colors

- Dark: `#1a1a2e` — 텍스트, 강한 contrast
- Light: `#faf9f5` — 배경 (사용 안 할 수도)
- Neutral Mid: `#8a8780` — 보조 텍스트, grid
- Neutral Light: `#e8e6dc` — 배경 분할, 비활성 막대

## Accent Colors (3-rotation)

- Accent Warm: `#d97757` — 첫 번째 강조 (coral/orange)
- Accent Cool: `#6a9bcc` — 두 번째 강조 (soft blue)
- Accent Natural: `#788c5d` — 세 번째 강조 (olive/green)
- Accent Red: `#c93636` — 임계점/위험/하이라이트
- Accent Purple: `#7b1fa2` — 보조 (자주 안 씀)

## Typography

- Heading: Malgun Gothic / NotoSansKR Bold — 22~24pt (figure 제목)
- Sub-heading: 14~15pt (panel 제목)
- Body / labels: 12~14pt
- Big numbers (stat callout): 40~46pt Bold

## Insight Box

- Background: `#fff3e0`
- Border: `#f57c00`
- Font: 12pt Bold

## Optimal Applications

- 데이터 분석 보고서 (Markdown / PDF 임베드)
- 노션 페이지 임베드
- 사내 발표 자료
- 친화적 톤 + 정량 데이터 의도 명확
