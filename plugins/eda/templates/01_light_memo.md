# Light Memo — NARROW Q&A 답변 템플릿

**Use when**: TOP N · 분포 · 평균 · 카운트 · 단일 통계 시각화 같은 single-shot 질문. 1-2개 Bash로 답이 나오는 경우.

**핵심 원칙 (Toss PANDA + Watcha 자료 통합)**:
사용자 EDA 자료의 모든 Key Findings 표는 항상 `지표 / 수치 / 시사점` 3컬럼 패턴.
→ NARROW 답변도 표만 던지지 말고 **반드시 1-2개 짧은 인사이트** 첨부. 단 질문 외 곁가지 X.

```markdown
🐼 [질문 1줄 요약]

[핵심 표 또는 분포 — 묻은 통계 그대로]

![<설명>](/eda-files/<filename>.png)   ← 시각화 요청 받았을 때만

📊 집계 기준
- 데이터: <data_path>
- 기간: <YYYY-MM-DD ~ YYYY-MM-DD> (KST, N일)
- 행 수: <N>건
- 스케일: 1-N {정수/실수} (dtype 기반 자동 판별)
- (있으면) 메타 enrichment: <movies.parquet / users.dat / ...>

💡 인사이트 (1-2개, 반드시)
- **<한 줄 시사점>** — <짧은 부연 (수치 + 도메인 맥락)>
- **<2번째>** (있을 때만)
```

## 인사이트 1-2개 선정 가이드

표만 봐서 사용자가 놓치기 쉬운 것 위주:

| 후보 (택 1-2) | 예시 |
|---|---|
| **데이터 함정** (필터링 인공물 / 스케일 오해) | "이 데이터는 ≥20 평점 유저만 포함 — 실 서비스의 cold-start 유저가 빠져있음" |
| **비교 baseline** (평소 대비 / 다른 그룹 대비) | "Drama 평균 ★3.77 vs 전체 ★3.58 — 0.19점 높음" |
| **롱테일 / 집중도** | "상위 5%가 전체의 28.4% — Pareto 80-20 약함" |
| **이상치 영향** | "1명이 전체 매출의 5.3% — 분석 시 제거 검토" |
| **도메인 맥락** | "왓챠 별점 1-10 척도 (1-5 아님) — 외부 데이터 비교 시 주의" |
| **모델링 권장** (DS audience 일 때) | "Class imbalance 심함 — ordinal loss 또는 class weight" |

**금지**:
- 표 수치 그대로 다시 적기 (예: "TOP1은 American Beauty 3,428건" — 표에 이미 있음)
- "데이터가 흥미롭다", "다양한 패턴" 같은 hedge
- "추가로 X도 분석해볼 수 있다" 같은 권장사항 (사용자가 묻지 않음)

## 시각화 동반 시 (사용자가 "그려줘" 요청)

- matplotlib 으로 `/tmp/eda/<filename>.png` 저장
- 답변 markdown에 `![](/eda-files/<filename>.png)` inline 첨부
- 한국어 폰트: `plt.rcParams['font.family']='NanumGothic'`
- 차트 1개당 파일 1개 (subplot 합치지 말 것)
- 막대그래프는 vertical (x=카테고리, y=수치)
- Watcha 테마 색상: TOP 3 강조 `['#d97757','#6a9bcc','#788c5d']`, 비강조 `#e8e6dc`

## 답변 예시

```markdown
🐼 ML-1M 평점 분포 (1-5 정수 별점)

| 별점 | 횟수 | 비율 |
|---|---:|---:|
| ★1 | 56,174 | 5.6% |
| ★2 | 107,557 | 10.8% |
| ★3 | 261,197 | 26.1% |
| ★4 | **348,971** | **34.9%** ← 최다 |
| ★5 | 226,310 | 22.6% |

📊 집계 기준
- 데이터: `data/rating_prediction/ml-1m/ratings.ftr`
- 기간: 2000-04-26 ~ 2003-03-01 (KST, 1,038일)
- 행 수: 1,000,209건
- 스케일: 1-5 정수 별점 (value_describe.max=5 + value.dtype=int)

💡 인사이트
- **강한 positivity bias** — ★3 이상 83.6%. 평균 3.58 / 중앙값 ★4 / IQR 사실상 0. 회귀 모델이 ★4 예측에 과적합될 위험.
- **ML-1M은 활성 평가자만 포함** (≥20 평점 필터링) — 실 서비스의 평점 주저 유저 패턴은 이 데이터에 없음.
```
