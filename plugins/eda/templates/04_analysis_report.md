# ML Data Analysis Report 템플릿

**Use when**: 개인 분석 노트, 미완료 분석 정리, "이 데이터로 X 분석해줘" 같이 분석 목표가 명확한 mid-size 작업. Full EDA 보다 가볍고 Light Memo 보다 깊이.

**기준 자료**: 사용자가 직접 정리한 ML Data Analysis Report 양식 (개인 메모장).

---

## 답변 구조 (markdown)

```markdown
# {분석할 데이터 or 모델명}

## 요약

- {핵심 결론 1줄}
- (선택) {중요한 발견 1-2개 bullet}

---

## 분석 목표

- {분석 목적 1-2 문장}
```

---

## 작성 원칙 (사용자 메모장 양식 기반)

- **분석 목표 명시** — "왜 이걸 분석하는가" 한 문장
- **데이터 소스별 메타 (기간 / 경로) 명시** — 재현 가능하게
- **질문 list 먼저** — 본문 분석 전에 "이걸 답하고 싶다" 나열
- **현재 활용 상태 + 확인하고 싶은 것** 분리 — 현 상태 vs 목표 명확
- **TOP N 케이스는 Appendix 로 분리** — 본문은 가볍게, 디테일은 Appendix
- **결론은 분석 목표 단계의 질문에 1:1 대응** — closure

---

## Light Memo 와의 차이

| | Light Memo | Analysis Report |
|---|---|---|
| 분량 | 30-50줄 | 100-200줄 |
| 데이터 소스 | 1개 | 1-3개 |
| 질문 | 1-2개 | 3-7개 |
| Appendix | 없음 | TOP N 등 |
| 목적 | 빠른 답 | 정리된 분석 |
